Questions

Questions Answers
What does the server in client server architecture do? The server responds to requests from the client.
What does the client do? The client makes requests through the browser.
What is an ip address? An unique number that identifies a computer or a device on a network.
What is DNS? DNS is like a phone book for the internet. It makes it easier for us to browse the web as we dont have to remember the IP-adresses, only the domain.
What is the difference between internet and WWW? The internet is a network of computers that are connected to eachother, and WWW is a service that runs on the network which you can use to browse web pages.
Explain what HTML is? HTML is a markup language, which is used to create the structure of a web page. It tells your browser what elements to display.
